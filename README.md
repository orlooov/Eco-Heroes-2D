# 🌱 Eco Heroes 2D

**Eco Heroes** — это мой прототип 2D-игры на движке Unity, посвященный теме защиты экологии. Проект создавался для изучения основ 2D-разработки, работы с физикой, анимациями и базовыми игровыми механиками.

## 🛠️ Стек технологий
- **Движок:** Unity (2D)
- **Язык программирования:** C#
- **Особенности:** Настройка 2D-физики, работа со спрайтами, система диалогов, инвентарь и UI.

## 👨‍💻 О проекте для резюме
Данный репозиторий содержит исходный код проекта. Это демонстрационная версия (MVP), которая показывает мои текущие навыки работы в среде Unity, верстки интерфейсов и написания скриптов на C#.
